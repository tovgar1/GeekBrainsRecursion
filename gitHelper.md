# Подсказка по GIT
## Создание репозитория

```sh
git init
```
##  Добавление файлов к коммиту
```sh
git add
```
##  Коммит (запись изменений)
```sh
git commit -m "Message"
```
##  Просмотр истории коммитов
```sh
git log
git log --oneline
```

## Возврат на прошлые коммиты
```sh
git checkout <commit_id>
```
